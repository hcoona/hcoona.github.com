и®әж–Үз¬”и®°пјҡ[FAST'03] ARC: A Self-Tuning, Low Overhead Replacement Cache

еҸ‘иЎЁдәҺ 2019-03-30 жӣҙж–°дәҺ 2023-08-07 еҲҶзұ»дәҺ Paper Note

ARCжҳҜдёҖз§Қзј“еӯҳжӣҝжҚўз®—жі•пјҢеңЁеҫҲеӨҡз§ҚиҙҹиҪҪзҺҜеўғзҡ„иЎЁзҺ°дјҳдәҺеёёз”Ёзҡ„LRUз®—жі•пјҢе№¶дё”е®һзҺ°йҡҫеәҰе’Ңз®—жі•еӨҚжқӮеәҰдёҺLRUиҝ‘дјјгҖӮ

ARCз®—жі•е…·жңүд»ҘдёӢдјҳиүҜзү№жҖ§пјҡ

  1. еңЁrecencyе’Ңfrequencyд№Ӣй—ҙжҢҒз»ӯзҡ„иҝӣиЎҢеҠЁжҖҒпјҲеңЁзәҝпјүи°ғж•ҙ

  2. ж— йңҖдәӢе…ҲжҢҮе®ҡзү№еҲ«зҡ„еҸӮж•°пјҲе…ҲйӘҢзҹҘиҜҶпјү

  3. е…·жңүе…ЁеұҖдјҳеҢ–зӯ–з•ҘпјҲж„ҸиҜ‘пјҢдёҚзЎ®е®ҡзҝ»иҜ‘зҡ„еҜ№дёҚеҜ№пјҢеҺҹж–Үempirically universalпјҢnoteиҜҙжҳҺиҜҘиҜҚеҮәиҮӘLZ77зҡ„и®әж–Үпјү

  4. еҸҜд»ҘпјҲеңЁжҹҗз§ҚзЁӢеәҰдёҠпјүжҠөжҠ—зәҝжҖ§жү«жҸҸпјҲscan-resistantпјү

еҺҹи®әж–ҮдёӯиҠұиҙ№дәҶеҫҲеӨҡзҜҮе№…д»Ӣз»Қд№ӢеүҚдәәд»¬иҜ•еӣҫжҸҗеҮәжҜ”LRUжӣҙеҘҪзҡ„з®—жі•зҡ„дёҖдәӣе·ҘдҪңпјҢе№¶иҝӣиЎҢдәҶдёҖз•ӘеҜ№жҜ”пјҢжң¬ж–ҮйҷҗдәҺзҜҮе№…е’ҢзІҫеҠӣз•ҘиҝҮиҝҷйғЁеҲҶеҶ…е®№гҖӮ

еҹәжң¬жҖқжғі

LRUзҡ„й—®йўҳеңЁдәҺе…¶еҸӘиҖғиҷ‘дәҶrecencyиҖҢе®Ңе…ЁжІЎжңүиҖғиҷ‘frequencyгҖӮдёҖиҲ¬иҖҢиЁҖпјҢдҝқеӯҳfreqencyдҝЎжҒҜзҡ„д»Јд»·еҸҲжҜ”иҫғеӨ§гҖӮARCзҡ„жңҖж №жң¬зҡ„жҖқи·ҜпјҲд»ҘдёӢз§°дёәDBLпјүеңЁдәҺдҪҝз”Ё2дёӘLRUпјҢе…¶дёӯдёҖдёӘ$L_1$еӯҳж”ҫжңҖиҝ‘иў«и®ҝй—®1ж¬Ўзҡ„ж•°жҚ®пјҲд№ҹе°ұжҳҜдј з»ҹзҡ„LRUпјүпјҢеҸҰдёҖдёӘ$L_2$еӯҳж”ҫжңҖиҝ‘иў«и®ҝй—®2ж¬ЎеҸҠд»ҘдёҠзҡ„ж•°жҚ®гҖӮиҝҷдёҖиҝ‘дјјеҮҸе°‘дәҶз»ҙжҠӨfrequencyзҡ„жҲҗжң¬гҖӮжңҖз»Ҳзӣ®ж ҮжҳҜе°ҪеҸҜиғҪз»ҙжҢҒ$|L_1| = |L_2| = c$пјҢдҪҶжҳҜи®әж–Үдёӯе№¶жІЎжңүи§ЈйҮҠдёәд»Җд№ҲиҰҒиҝҷд№ҲеҒҡгҖӮпјҲжҖ»жҳҜиғҪдҝқжҢҒ$|L_1| + |L_2| = 2c$пјҢеҰӮжһң$|L_1| < c$ж—¶дјҡдјҳе…Ҳж·ҳжұ°$L_2$дёӯзҡ„еҶ…е®№з»ҷ$L_1$и…ҫең°е„ҝпјү

жҲ‘д»¬жҠҠж•ҙдёӘCacheеҲҶдёәд»ҘдёӢ2дёӘйғЁеҲҶпјҡ

  1. Cache Directory

  2. Cache Item

е…¶дёӯCache Directoryз”ЁдәҺиҝӣиЎҢзҙўеј•пјҢCache ItemзңҹжӯЈиЎЁзӨәеңЁзј“еӯҳдёӯзҡ„еҶ…е®№гҖӮеҜ№дәҺдј з»ҹLRUз®—жі•иҖҢиЁҖпјҢд»ҘдёҠ2иҖ…зҡ„еӨ§е°ҸжҳҜдёҖж ·зҡ„пјҲDirectoryзҙўеј•зҡ„йЎ№зҡ„ж•°йҮҸе’ҢзңҹжӯЈеңЁзј“еӯҳдёӯзҡ„еҶ…е®№зҡ„ж•°йҮҸпјүгҖӮеҜ№дәҺдёҠйқўжҸҗеҲ°зҡ„DBLиҖҢиЁҖпјҢд»ҘдёҠ2иҖ…зҡ„еӨ§е°Ҹд№ҹжҳҜдёҖж ·зҡ„пјҢдёә$|L_1| + |L_2| = 2c$гҖӮзү№еҲ«зҡ„пјҢеңЁDBLдёӯпјҢ$L_1$е°ұжҳҜдёҖдёӘдј з»ҹзҡ„LRUпјҢе…¶еӨ§е°Ҹдёә$c$гҖӮ

дёҠйқўжҸҗеҲ°зҡ„DBLз®—жі•зҡ„й—®йўҳжҳҜдҪҝз”ЁдәҶ2еҖҚдәҺLRUзҡ„з©әй—ҙпјҢе…¶ж•ҲжһңжҜ”LRUеҘҪжҳҜжҳҫз„¶зҡ„пјҢиғҪдёҚиғҪдҪҝз”Ёе’ҢLRUдёҖж ·еӨҡзҡ„з©әй—ҙеҫ—еҲ°жӣҙеҘҪзҡ„ж•Ҳжһңе‘ўпјҹдёҖдёӘзӣҙжҺҘзҡ„жғіжі•жҳҜдҪҝз”Ё$2c$зҡ„Cache DirectoryдҪҶжҳҜеҸӘдҝқз•ҷ$c$дёӘCache Itemеёёй©»еҶ…еӯҳпјҢйӮЈд№ҲеңЁиҝҷ$2c$дёӘзҙўеј•йЎ№дёӯеҰӮдҪ•еҸ–иҲҚе°ұжҳҜдёҖдёӘй—®йўҳгҖӮеҰӮ[figure-1]жүҖзӨәпјҢжҲ‘д»¬е°Ҷ$L_1$е’Ң$L_2$еҲҶеҲ«жӢҶеҲҶдёә$T_1, B_1, T_2, B_2$гҖӮзӣҙи§үдёҠпјҢ$T_1$жҜ”$B_1$жӣҙжңүд»·еҖјпјҢ$T_2$жҜ”$B_2$жӣҙжңүд»·еҖјпјҢеҰӮжһңжҲ‘д»¬иғҪеӨҹз»ҙжҢҒ$|T_1| + |T_2| \le c$пјҢеҲҷеә”еҪ“е°Ҷ$T_1 \cup T_2$дёӯзҡ„е…ғзҙ дҝқз•ҷеңЁCacheдёӯгҖӮ

еү©дёӢзҡ„й—®йўҳпјҢеңЁдәҺеҰӮдҪ•еңЁ$L_1 \cup L_2$дёӯи°ғиҠӮ$T_1 \cup T_2$зҡ„дҪҚзҪ®пјҲеҚіи°ғиҠӮ$|B_1|$е’Ң$|B_2|$пјүпјҢд»ҘеҸҠеҰӮдҪ•еңЁ$T_1 \cup T_2$дёӯи°ғиҠӮдёӨиҖ…зҡ„еӨ§е°ҸеҲҶй…ҚгҖӮзү№еҲ«зҡ„пјҢеҪ“$|L_1| = |T_1| = c$ж—¶пјҢиҜҘз®—жі•зӯүд»·дәҺLRUпјҢд№ҹе°ұжҳҜиҜҙеҰӮжһңи°ғиҠӮеҫ—еҪ“зҡ„иҜқпјҢиҜҘз®—жі•иҮіе°‘иғҪеҒҡеҲ°е’ҢLRUдёҖж ·еҘҪгҖӮ

ARCз®—жі•

з”ұдәҺжҲ‘д»¬иҰҒжұӮдҝқжҢҒжҖ§иҙЁ$|L_1| = |L_2| = c$пјҢеӣ жӯӨеҪ“и°ғиҠӮ$T_1$е’Ң$T_2$зҡ„е…ізі»ж—¶пјҢе®һйҷ…дёҠе°ұи°ғиҠӮдәҶ$T_1 \cup T_2$еңЁ$L_1 \cup L_2$дёӯзҡ„дҪҚзҪ®гҖӮзӣҙи§үдёҠзӣёеҪ“дәҺеңЁ$L_1$е’Ң$L_2$дёӯй—ҙзҡ„ең°ж–№жңүдёҖдёӘеӣәе®ҡеӨ§е°Ҹзҡ„ж»‘еқ—пјҢжҲ‘д»¬еңЁи°ғиҠӮиҝҷдёӘж»‘еқ—зҡ„дҪҚзҪ®гҖӮдёҚеҰЁд»Ө$|T_1| = p$пјҢеҲҷ$|B_1| = |T_2| = c - p$дё”$|B_2| = p$гҖӮ

жҺҘдёӢжқҘпјҢжҲ‘д»¬йңҖиҰҒжҖқиҖғдёҖдёӢ$B_1$е’Ң$B_2$еҜ№жҲ‘д»¬иҖҢиЁҖж„Ҹе‘ізқҖд»Җд№ҲгҖӮеӣһжғідёҠйқўжҸҗеҲ°зҡ„DBLпјҢ$L_1$ж„Ҹе‘ізқҖжңҖиҝ‘еҸӘе‘ҪдёӯдәҶ1ж¬Ўзҡ„зј“еӯҳйЎ№зҙўеј•пјҢе…¶дёӯ$T_1$жҳҜжҲ‘д»¬дҝқз•ҷеңЁзј“еӯҳдёӯзҡ„еҶ…е®№пјҢ$B_1$жҳҜжҲ‘д»¬жІЎжңүдҝқз•ҷеңЁзј“еӯҳдёӯзҡ„еҶ…е®№гҖӮеӣ жӯӨпјҢеҰӮжһңжҹҘиҜўиҜ·жұӮе‘ҪдёӯдәҶ$B_1$пјҢж„Ҹе‘ізқҖжҲ‘д»¬еә”еҪ“жҠҠиҝҷж¬Ўе‘ҪдёӯеҪ“дҪңе‘ҪдёӯдәҶ$L_1$жқҘзңӢпјҢе№¶дё”зӣёеә”зҡ„пјҢиҜҙжҳҺжҲ‘д»¬зҡ„$|T_1|$еҸҜиғҪе°ҸдәҶпјҢжІЎиғҪе°Ҷиҝҷж¬Ўе‘Ҫдёӯзҡ„ж•°жҚ®еӯҳиө·жқҘгҖӮи®әж–ҮдёӯпјҢеҪ“$|B_1| \gt |B_2|$ж—¶пјҢ$p$еўһеҠ 1пјҢеҪ“$|B_1| < |B_2|$ж—¶пјҢ$p$еўһеҠ $|B_2| / |B_1|$пјҢдҪҶжҳҜи®әж–ҮдёӯжІЎжңүи§ЈйҮҠдёәд»Җд№Ҳиҝҷд№Ҳи®ҫи®ЎгҖӮ

е®Ңж•ҙзҡ„ARCз®—жі•еҰӮ[figure-2]жүҖзӨәпјҢе…¶дёӯCase I-IIIеҜ№еә”дәҺDBLе‘Ҫдёӯзҡ„жғ…еҶөпјҢCase IVеҜ№еә”дәҺDBLжңӘе‘Ҫдёӯзҡ„жғ…еҶөгҖӮ

й—®йўҳеӣһйЎҫ

д№ӢеүҚжҸҗеҲ°дәҶдёҖдәӣй—®йўҳпјҢеңЁжӯӨз»§з»ӯжҺўи®ЁдёҖдёӢгҖӮ

$L_1$е’Ң$L_2$зҡ„еӨ§е°ҸжҳҜеҗҰеҸҜд»Ҙи°ғж•ҙ

еңЁзӣ®еүҚзҡ„и®ҫи®ЎдёӯпјҢ$|L_1| = |L_2| = c$пјҢз”ұдәҺжҲ‘д»¬жңҖеӨҡеҸӘдҝқз•ҷ$|T_1| + |T_2| = c$йЎ№еҶ…е®№пјҢеңЁжһҒз«Ҝжғ…еҶөдёӢ$|L_1| = |T_1| = c$пјҢжӯӨж—¶иҝӣдёҖжӯҘеўһеӨ§$L_1$д№ҹдёҚиғҪеҶҚиҝӣиЎҢд»Җд№Ҳи°ғж•ҙдәҶпјҲеҸ—йҷҗдәҺCacheеӨ§е°ҸпјүпјҢд»ҺиҝҷдёӘи§’еәҰжқҘзңӢеңЁ$L_1$е’Ң$L_2$д№Ӣй—ҙиҝӣиЎҢи°ғж•ҙеҸҜиғҪжҳҜжІЎд»Җд№Ҳж„Ҹд№үзҡ„гҖӮ

дҪҶжҳҜеңЁи®әж–Үдёӯзү№еҲ«жҸҗеҲ°дәҶ E. Extra HistoryпјҢи®Ёи®әдәҶиҝҷж ·дёҖдёӘй—®йўҳпјҡ

An interesting question is whether we can incorporate more history information in ARC to further improve it.

и®әж–Үдёӯз»ҷеҮәдәҶдёҖз§ҚеҲ©з”ЁйўқеӨ–зҡ„з©әй—ҙзҡ„ж–№жі•пјҢдјјд№Һжҡ—зӨәдәҶиҝҷд№ҲеҒҡжҳҜжңүз”Ёзҡ„пјҢдҪҶжҳҜжІЎжңүи®әиҜҒеҜ№жҜ”иҝҷдёҖж–№жі•зҡ„ж•ҲжһңеҰӮдҪ•гҖӮ

$p$зҡ„и°ғж•ҙдёәд»Җд№ҲдёҚжҳҜеӣәе®ҡзҡ„

жҺЁжөӢжҳҜеӣ дёәеҪ“$B_1$е°Ҹзҡ„ж—¶еҖҷе‘Ҫдёӯ$B_1$жҳҜжҜ”иҫғеӣ°йҡҫзҡ„пјҢиҰҒеҜ№жӯӨеҒҡдёҖдёӘеҠ жҲҗгҖӮдҪҶжҳҜдёәд»Җд№Ҳиҝҷд№ҲеҒҡи®әж–Үе№¶жІЎжңүз»ҷеҮәдёҖдёӘи§ЈйҮҠпјҢд№ҹжІЎжңүд»Җд№ҲеҜ№жҜ”гҖӮ

Scan-Resistant

зӣҙи§үдёҠARCжҳҜеҸҜд»ҘжҠөеҫЎжү«жҸҸPatternзҡ„иҜ·жұӮзҡ„пјҢеӣ дёәARCдёҚд»…иҖғиҷ‘дәҶrecencyпјҢиҝҳиҖғиҷ‘дәҶfrequencyгҖӮе…·дҪ“иҖҢиЁҖпјҢжңүд»ҘдёӢ2дёӘеҺҹеӣ пјҡ

  1. жү«жҸҸдјҡеҲ·ж–°$L_1$зҡ„еҶ…е®№пјҢдҪҶжҳҜдёҚдјҡеҪұе“Қ$L_2$зҡ„еҶ…е®№

  2. жү«жҸҸдјҡжӣҙеӨҡзҡ„е‘Ҫдёӯ$B_2$пјҲзӣёиҫғдәҺ$B_1$пјүпјҢеӣ жӯӨдјҡиҝӣдёҖжӯҘеҮҸе°Ҹ$|T_1|$пјҢиҝӣиҖҢдҪҝеҫ—жү«жҸҸйҖ жҲҗзҡ„еҪұе“ҚжҜ”еҚ•зәҜиҖғиҷ‘з¬¬1зӮ№жӣҙе°Ҹ

дёҚжҖ»з»“жҖ»и§үеҫ—е°‘зӮ№д»Җд№Ҳ

жҖ»зҡ„жқҘиҜҙпјҢARCз®—жі•йҖҡиҝҮдёҖз§ҚжҜ”иҫғе·§еҰҷзҡ„ж–№жі•пјҢеңЁдёҚжҳҫи‘—еўһеҠ е®һзҺ°жҲҗжң¬е’Ңз®—жі•ж—¶й—ҙеӨҚжқӮзҡ„жғ…еҶөдёӢпјҢиҫғдёәжҳҫи‘—зҡ„ж”№иҝӣдәҶLRUз®—жі•гҖӮдҪҝз”ЁLRUжқҘжүҝиҪҪжңҖиҝ‘йҒҮеҲ°2ж¬Ўд»ҘдёҠзҡ„ж•°жҚ®жқҘиҝ‘дјјжҚ•иҺ·frequencyдҝЎжҒҜжҳҜдёҖеӨ§дә®зӮ№гҖӮеҠЁжҖҒе№іиЎЎзҡ„ж–№жЎҲзӣҙи§үдёҠжңүж•ҲпјҢдҪҶжҳҜз»ҶиҠӮдёҠзјәд№ҸжңүеҠӣзҡ„и®әиҜҒпјҢдёҚзЎ®е®ҡжҳҜдёҚжҳҜдёҖдёӘжңҖдјҳзҡ„з®—жі•гҖӮеҜ№дәҺеҠЁжҖҒеҸҳеҢ–зҡ„ж•°жҚ®иҖҢиЁҖпјҢеҠЁжҖҒе№іиЎЎзҡ„з®—жі•дјҳдәҺйқҷжҖҒзҡ„з®—жі•жҳҜеҸҜд»Ҙйў„жңҹзҡ„гҖӮ
